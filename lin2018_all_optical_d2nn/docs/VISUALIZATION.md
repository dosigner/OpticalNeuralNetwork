# Visualization Guide

All plotting functions must:
- use physical axis labels in `mm`
- include colorbars and deterministic style
- support `save_path`, `show`, and `title`

Required outputs:
- phase masks by layer
- output intensity with detector overlay
- confusion matrix
- detector energy heatmap
- imaging comparison (input, D2NN, free-space)
- propagation stack/cross-section
