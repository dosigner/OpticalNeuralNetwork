# Code Tree

The repository follows the structure requested in `spec.md`.

- `src/d2nn/physics`: grid, ASM, material transforms
- `src/d2nn/models`: D2NN layers and constraints
- `src/d2nn/detectors`: region layout and energy integration
- `src/d2nn/training`: losses, loops, callbacks, sweeps
- `src/d2nn/viz`: deterministic figure generation
- `src/d2nn/export`: height maps and Lumerical integration
- `src/d2nn/cli`: runnable commands
- `tests/`: unit and smoke tests
- `configs/`: reproducible experiment configs
